About the
Project

CineANN is a research-driven movie recommendation system that bridges the gap between deep learning and cinema.

Mission

Our goal was to create a personalized cinematic experience where technology disappears, leaving only the perfect recommendation. By leveraging Artificial Neural Networks, we analyze thousands of latent movie features to find the hidden patterns in what you love.

Technology

Built with a Python/Flask backend and a custom pure-NumPy inference engine, CineANN is optimized for speed and stability. The frontend utilizes a unique vintage-inspired design system we call "Lumina Noir."

Mathematical Logic

The system treats recommendation as a Regression Problem. Instead of just finding similar movies, the ANN tries to predict the precise rating a user would give to a movie they haven't seen.

1. Embedding Space

Users and Movies are mapped into a high-dimensional latent space. Two points close to each other in this space indicate a high probability of a positive interaction.

2. Non-Linear Patterns

By using ReLU (Rectified Linear Unit) activation functions in the hidden layers, the network can learn complex, non-linear relationships that traditional matrix factorization misses.

3. Pure NumPy Inference

To ensure lightning-fast performance, the production engine uses a zero-dependency NumPy implementation of the weights, avoiding the overhead of heavy frameworks like TensorFlow during live use.
