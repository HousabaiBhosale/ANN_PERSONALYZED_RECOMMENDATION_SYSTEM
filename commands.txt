# ANN Personalized Recommendation System - Commands

## Prerequisites
- Python 3.8+
- pip (Python package manager)

## Installation Steps
1. Clone the repository (if not already done):
   git clone https://github.com/HousabaiBhosale/ANN_PERSONALYZED_RECOMMENDATION_SYSTEM
   cd ANN_PERSONALYZED_RECOMMENDATION_SYSTEM

2. Create a virtual environment (Optional but recommended):
   python -m venv venv
   # On Windows:
   .\venv\Scripts\activate
   # On macOS/Linux:
   source venv/bin/activate

3. Install dependencies:
   pip install -r requirements.txt

## Running the Project
The frontend and backend are integrated into a single Flask application.

1. Start the Flask server:
   python app.py

2. Open your web browser and navigate to:
   http://localhost:5000

## Backend Commands (Internal API usage)
The server runs on port 5000. You can interact with it via POST requests to:
- Predict: POST /predict (JSON: {"userId": 1, "movieId": 31})
- Recommend: POST /recommend (JSON: {"userId": 1, "n": 10})
- Explain: POST /explain (JSON: {"userId": 1, "movieId": 31})
- Benchmark: GET /benchmark
